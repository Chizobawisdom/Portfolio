Ziel: Automatische Wurzelbestimmung für beliebige Polynome (n ≥ 2) ohne kommerzielle Frameworks.

Vorgehen: Iteratives Newton-Raphson-Verfahren mit explizitem Horner-Schema, automatisiertem Ableitungsmodul (dpoly) und Konvergenzschutz auf 1e-6 Genauigkeit.

Ergebnis: Schnelles, eigenständiges numerisches Skript mit Wurzelvalidierung und Funktionskurven-Darstellung via matplotlib.

Newton-Iterations-Kern 
def solver(xi, coeff):
    maxError = 1e-6
    error = math.inf
    while (error > maxError):
        df = poly(xi, dpoly(coeff))
        if (df != 0):
            xn = xi - poly(xi,coeff)/df
        else:
            xn = xi - poly(xi,coeff)/\
                 random.uniform(-1e-12,1e-12)
        error = abs(xn - xi)
        xi = xn
    return format(xn, '.4f')
