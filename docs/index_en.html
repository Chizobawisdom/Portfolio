Histogram Matching from Scratch — Arithmetic Approach

Objective: Reproduce histogram specification without relying on high-level library functions — building the full matching pipeline manually using only NumPy to demonstrate deep understanding of the underlying mathematics.

Process: Computed pixel intensity histograms for both the dark source and high-contrast reference using np.bincount(). Normalised each to a probability density function (PDF) and accumulated into a cumulative distribution function (CDF). Built a 256-entry lookup table by finding, for each source intensity level, the reference intensity whose CDF value is closest — minimising |CDF_source(i) − CDF_reference(j)| via np.argmin(). Applied the mapping via direct array indexing (mapping[image]) for a fully vectorised transform. Results visualised as a 2×3 grid of image previews and their corresponding histograms.

Outcome: A transparent, dependency-minimal implementation that makes the statistical mechanics of histogram specification explicit — reinforcing the same result achieved via match_histograms() but with full interpretability over each computational step.
