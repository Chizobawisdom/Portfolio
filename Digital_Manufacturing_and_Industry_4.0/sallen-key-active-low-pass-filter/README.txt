 # LT Spice Simulation
 Description:
A second-order Sallen-Key active low-pass filter designed for frequency response analysis. Key features:

---
Applications
 - Signal conditioning Sensor & Measurement Systems
 - Audio Signal processing
 - Rapid Prototyping for Embedded & Instrumentation Designs

---
Topology: Sallen-Key configuration using a single op-amp (U2)

Component Values: 47K and 2.2K resistors with 440nF and 22nF capacitors

Cutoff Frequency: Approximately 16.4 kHz

Input Termination: 50Ω source resistance for proper impedance matching

---

Simulation: AC analysis from 1 Hz to 500 kHz (10 points per decade)
